Place image, audio, font, and data resources here.

Recommended folders:

- `bg/` for backgrounds.
- `chara/` for character sprites.
- `bgm/` for music.
- `voice/` for voice files.

Project Suzu registers resource ids from paths without extensions, such as `bg/title_bg` for `assets/bg/title_bg.png`. Texture files are also available by their file stem, such as `title_bg`, for short script commands.

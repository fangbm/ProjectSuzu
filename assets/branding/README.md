# Project Suzu Branding

The files in this directory are Project Suzu project identity assets.

- `Suzu_icon.png` is the release/package icon.
- `Suzu_icon_square.png` is a square variant for launchers and stores.
- `Suzu_icon.ico` is the Windows icon bundle.

You may use these assets when identifying an unmodified Project Suzu distribution, documentation page, local tool, or example package. Do not use them to imply endorsement of a fork, plugin, asset pack, or game project unless that relationship is explicit.

External games built with Project Suzu should provide their own branding in release packages.

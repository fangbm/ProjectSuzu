# Project Suzu 品牌素材

本目录中的文件是 Project Suzu 项目身份素材。

- `Suzu_icon.png` 是 release/package 图标。
- `Suzu_icon_square.png` 是用于 launcher 和 store 的正方形变体。
- `Suzu_icon.ico` 是 Windows icon bundle。

你可以在标识未修改的 Project Suzu 发行版、文档页面、本地工具或示例包时使用这些素材。不要用它们暗示某个 fork、plugin、素材包或游戏项目得到了官方背书，除非这种关系已经明确说明。

使用 Project Suzu 构建的外部游戏应在自己的发布包中提供独立品牌素材。
